# Cyber-OS-Academy
Hands-on workshop project visualizing C-SCAN and SJF operating system algorithms.

An interactive web-based project that explains core **Operating System algorithms**
through visual and beginner-friendly content.

**Live Project:**  
https://cyber-os-academy.lovable.app/

## Algorithms Covered
- C-SCAN Disk Scheduling
- Shortest Job First (SJF) CPU Scheduling

## Tech Stack
- Lovable (AI-assisted web development)

## Project Context
Created as part of a **hands-on workshop**, where different groups worked on
different combinations of OS algorithms.  
This project was **conceptualized and structured by me**.

## AI Usage
AI tools were used to assist development and improve explanations.
